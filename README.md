## Compilation

On *nix systems, you can compile the project using the following commands:
(the list of dependencies is abstract, so good luck figuring out)
```sh
cmake -S . -B build
cmake --build build
./build/nanotekspice Circuit.nts
```

Or if you have `nix`, just use `nix run`.
